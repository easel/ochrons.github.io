Correctness
Strong typing guarantees your code will run when it compiles. No more silly JavaScript mistakes due to mistyped function or field names. No accidental mixing of strings and numbers. Use Scala's powerful typing to create clean, maintainable code even in large applications.
Speed
Scala.js compiles Scala code into highly efficient JavaScript utilizing an internal optimizer. Incremental compilation guarantees speedy turn-around times when your code changes.
Interoperability
Use jQuery or any HTML5 feature directly from your Scala.js code, either in a typed or untyped way. All JavaScript libraries work with Scala.js code.
